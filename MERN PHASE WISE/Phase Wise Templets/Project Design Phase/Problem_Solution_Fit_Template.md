# Problem – Solution Fit Template
## Project: UCab – Cab Booking Application

---

## 1. Problem Definition

### Core Problem
People who need cab services face difficulty in finding available cabs quickly, comparing prices transparently, and managing their bookings in one place. Cab administrators have no digital tool to manage their fleet efficiently.

### Target Users
| User Type | Description |
|-----------|-------------|
| Regular User | Working professionals, college students who need daily cab services |
| Admin | Cab fleet owner or manager who manages drivers and cabs |

---

## 2. Problem – Solution Mapping

| # | Problem | Solution in UCab |
|---|---------|-----------------|
| 1 | No single platform to view all available cabs | UCab shows all available cabs on one page with images, details and pricing |
| 2 | Fare is unknown until trip ends | Live fare estimate shown before booking (distance × price per km) |
| 3 | Difficult to compare cab prices | Sort by price feature (Low to High / High to Low) |
| 4 | No booking history maintained | My Bookings page with complete trip history |
| 5 | Cannot cancel a booking easily | Cancel button on active bookings with confirmation |
| 6 | No admin tool to manage cab fleet | Admin panel with add, view, and manage cabs |
| 7 | No visibility into platform activity | Admin dashboard with live counts of Users, Cabs, Bookings |
| 8 | Security concerns with shared platforms | JWT authentication with role-based access control |

---

## 3. Value Proposition

**For Users:**
> UCab gives you instant access to all available cabs with upfront pricing, so you can book the right ride in seconds without any surprises.

**For Admins:**
> UCab gives you a complete digital control panel to manage your cab fleet, monitor bookings, and track users — all in one place.
