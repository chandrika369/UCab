## 7. Technology Justification

| Technology | Why Chosen |
|------------|-----------|
| React.js | Component-based, fast rendering, large ecosystem |
| Node.js | Same language (JS) on both frontend and backend |
| Express.js | Minimal, flexible, easy to set up REST APIs |
| MongoDB | Schema-flexible NoSQL, perfect for booking data |
| JWT | Stateless, secure, easy to implement role-based auth |
| Bootstrap 5 | Quick responsive UI without writing custom CSS |
| Vite | Faster than CRA, instant HMR for development |
