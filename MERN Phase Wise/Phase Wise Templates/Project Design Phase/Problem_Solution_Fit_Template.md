# Problem – Solution Fit
## Project: UCab – Cab Booking Application

---

## Problem – Solution Mapping

| Problem | Solution in UCab |
|---------|-----------------|
| No single platform to view all available cabs | UCab shows all available cabs on one page |
| Fare is unknown until trip ends | Live fare estimate (distance × price per km) |
| Difficult to compare cab prices | Sort by price feature |
| No booking history maintained | My Bookings page with complete trip history |
| Cannot cancel a booking easily | Cancel button on active bookings |
| No admin tool to manage cab fleet | Admin panel with add, view and manage cabs |
| No visibility into platform activity | Admin dashboard with live counts |
| Security concerns | JWT authentication with role-based access |

---

## Value Proposition

For Users:
UCab gives instant access to all available cabs with upfront pricing so you can book the right ride in seconds without any surprises.

For Admins:
UCab gives a complete digital control panel to manage your cab fleet, monitor bookings, and track users all in one place.
